Verwenden Sie beim Erstellen von Schnittstellen Ionic Framework-UI-Komponenten anstelle von benutzerdefiniertem HTML/CSS. Bevorzugen Sie Ion-Tags, verwenden Sie ionische CSS-Variablen für Farben, verwenden Sie Ionicons für Symbole und greifen Sie nur dann auf benutzerdefinierte Stile zurück, wenn kein ionisches Äquivalent vorhanden ist.
