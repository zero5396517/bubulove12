Используйте компоненты пользовательского интерфейса Ionic Framework вместо пользовательского HTML/CSS при создании интерфейсов. Отдавайте предпочтение тегам ion, используйте переменные Ionic CSS для цветов, используйте Ionicons для значков и возвращайтесь к пользовательским стилям только тогда, когда не существует эквивалента Ionic.
